Dos dados

Os dados do auxílio emergencial são primordiais para a execução deste projeto visto que eles são os insumos básicos referentes ao pagamento mensal do auxílio à população sobre os quais elaboramos todos os artefatos previstos. Aos dados já coletados anteriormente foi verificado que eles só precisavam de mais um conjunto de dados que relaciona os estados da federação às macrorregiões definidas em 1970 pelo Instituto Brasileiro de Geografia e Estatística - IBGE, são elas: Norte, Sul, Centro Oeste, Sudeste e Nordeste.

Os conjuntos de dados utilizados foram:

  • Auxílio Emergencial

    • Estes dados referem-se às informações do pagamento do auxílio emergencial mês a mês, no período de abril a agosto de 2020.

  • IPCA

    • O IPCA/IBGE mede a variação dos custos dos gastos conforme acima descrito no período do primeiro ao último dia de cada mês de referência. No período do dia onze ao dia vinte do mês seguinte o IBGE divulga as variações.

  • População

    • Os dados sobre a população brasileira versam sobre o total estimado da população brasileira pelo Instituto Brasileiro de Geografia e Estatística (IBGE) para o ano de 2020. Este conjunto de dados permitirá a comparação entre diferentes os diferentes estados pois tem-se então dados proporcionais entre eles.

  • Rendimento

    • Os dados sobre o rendimento dizem respeito a renda da população.

  • Regiões

    • Conforme dito, dados relacionam os estados e suas respectivas regiões.
